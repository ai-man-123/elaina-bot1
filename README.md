## Getting Started

### This project require NodeJS v14.
This is a Bot Group Type, so you have to enter the bot into a group in order to fully use the command!

### FFMPEG
- [DOWNLOAD](https://ffmpeg.org/)
- [TUTORIAL](https://youtu.be/04Gf6TEnmjk)

### IMAGEMAGICK
- [DOWNLOAD](https://imagemagick.org/script/download.php)
Note : Kalo Mau Work, Centang Kolom 1,2,3,5,6

### Install
Clone this project

```bash
> git clone https://github.com/ai-man-123/elaina-bot1
> cd elaina-bot1
```

Install the dependencies:

```bash
> npm install
> npm install -g sharp
> npm install -g pm2
```

### Usage
1. run the Whatsapp bot

```bash
> npm start
```
or
```bash
> pm2 start run.js
> pm2 monit
```

2. stop the Whatsapp bot

for npm :
```bash
> ctrl + c
```
for pm2 :
```bash
> pm2 stop
```
